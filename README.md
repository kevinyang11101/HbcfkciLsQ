# 基于Python的摄影平台交流系统 python1259

## 项目概述

本项目是一个基于Python开发的摄影平台交流系统，旨在为摄影爱好者提供一个作品展示、交流与分享的空间。

## 技术栈

- **后端框架**：Django
- **Python版本**：3.7/3.8
- **数据库**：MySQL 5.7 或更高版本（理论支持5.X系列）
- **数据库工具**：Navicat 11（推荐使用）
- **开发工具**：PyCharm、VS Code
- **前端框架**：Vue.js

## 功能模块

- 用户注册与认证
- 作品上传与展示
- 评论区交流互动
- 用户个人中心

## 系统角色

- 普通用户：注册、上传作品、评论互动等
- 管理员：系统管理、用户管理、作品审核等

## 运行环境

- Python 3.7/3.8
- MySQL 5.7 或更高版本
- Node.js（前端Vue.js运行环境）

## 部署步骤

1. 安装Python和MySQL环境
2. 使用pip安装Django及其他依赖库
3. 安装Node.js和npm，以便运行Vue.js
4. 部署后端Django项目
5. 部署前端Vue.js项目

## 目录结构

```
- backend                # 后端项目目录
  - app                  # 应用代码目录
  - static               # 静态文件目录
  - media                # 媒体文件目录
  - manage.py            # 管理脚本
- frontend               # 前端项目目录
  - src                  # Vue.js源码目录
  - public               # 公共资源目录
  - package.json         # Node.js项目配置文件
```

## 核心亮点

- 基于成熟的Django框架，具有高度的可扩展性和稳定性
- 支持多种版本的Python和MySQL，兼容性强
- 清晰的目录结构，便于项目管理和二次开发
- 丰富的功能模块，满足用户的不同需求
